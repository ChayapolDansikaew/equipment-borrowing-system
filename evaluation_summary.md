# 5. สรุปผลการประเมิน (Evaluation Summary)

ผลการทดสอบการประเมินการใช้งานและความสวยงามของหน้าเว็บโดยผู้ใช้สามารถสรุปได้ ดังนี้

- **หน้าเว็บส่วนใหญ่มีการทำงานที่ตรงไปตรงมาและไม่ซับซ้อน** โดยเฉพาะระบบการเข้าสู่ระบบผ่าน Chula SSO และการค้นหาอุปกรณ์ที่สามารถทำความเข้าใจได้ง่าย อย่างไรก็ตาม ในส่วนของกระบวนการจัดการตะกร้าและส่งคำขอยืม ผู้ใช้อาจต้องใช้เวลาทำความเข้าใจเงื่อนไขที่เข้มงวดของระบบเล็กน้อย ในขณะที่ฝั่งผู้ดูแลระบบ (Admin) มองว่าระบบจัดการค่าปรับและการอนุมัติคำขอค่อนข้างมีความซับซ้อนและต้องใช้ความระมัดระวังสูงในการตัดสินใจ
- **ผู้ใช้ที่เข้าร่วมการทดสอบรู้สึกพึงพอใจมากในด้านความสวยงามและการจัดวางองค์ประกอบ** โดยเฉพาะในหน้าแดชบอร์ดสถิติ (Dashboard) และหน้าจัดการคำขอที่ใช้รูปแบบ Kanban Board รวมถึงหน้าแรก (Home) ที่มีการใช้รูปแบบ Modern UI และการจัดวางรูปภาพอุปกรณ์ที่ชัดเจน ช่วยเพิ่มสีสันและความน่าสนใจ ทำให้การค้นหาและเรียกดูอุปกรณ์ง่ายและมีประสิทธิภาพขึ้น
- **ในส่วนของหน้าที่มีการใช้ตารางข้อมูล (Data Tables) เป็นหลัก** เช่น หน้าประวัติการทำรายการ หรือหน้าจัดการค่าปรับ ผู้ใช้ยังไม่ประทับใจมากนัก เนื่องจากข้อมูลที่มีปริมาณมากทำให้หน้าจอดูทึบและลายตา มีข้อเสนอแนะให้ปรับปรุงสีสันและรูปแบบของตารางให้มีความโปร่งตา ดึงดูด และช่วยให้อ่านข้อมูลจำนวนมากได้ใช้งานง่ายยิ่งขึ้น
